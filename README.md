### ¿Cómo puedo ejecutarlo?

Necesitas tener los paquetes `flex` y `g++` instalador.

```bash
$ sudo apt install flex g++
```

Después ejecuta el script `ejecutar.sh`
para compilar y ejecutar el programa.

```bash
% ./ejecutar.sh <fichero_de_texto>
```
